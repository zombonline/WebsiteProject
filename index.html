We played the hit new game Caloop!

An awesome brain teaser

Caloop is an awesome mathematical puzzle game. You are tasked with getting from 0 to a target number within a set amount of 'actions'. The game combines a pleasing aesthetic and a calm relaxing soundtrack to keep you in the zone while you figure out progressively more complex puzzles.
